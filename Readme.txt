--------------------------------BANK MANAGEMENT SYSTEM-----------------------------------


-------------------------------------------------------------------------------------------
------------------------------------INTRODUCTION----------------------------------------
-------------------------------------------------------------------------------------------

This is a management system which allows the admin of a bank to create , remove and handle
access of the employees on different functions like withdraw , deposit , create and delete.
Similarly bank accounts for customers are opnened by employees by giving credentials to the
system like user cnic and password. And then the user can sign up and login to the bank app
through the password that has beeen kept for his account.

-------------------------------------------------------------------------------------------
------------------------------------Prequistites-------------------------------------------
-------------------------------------------------------------------------------------------
1.Java Setup on your machine.
    1.1 Install Microsoft jdk v19.
    1.2 Add the /bin folder of your installation to the PATH system variable.
    1.3 Open a command prompt and type java -version to see if java is installed correctly.
2.MySql Setup on your machine.
    2.1 Install MySql on your machine.
    2.2 Set the MySql_HOME system variable to the /bin folder of your installation.
    2.3 Add the /bin folder of your installation to the PATH system variable.
    2.4 Open a command prompt and type mysql -version to see if mysql is installed correctly.
3.SceneBuilder Setup on your machine.
    3.1 Install SceneBuilder setup.
    3.2 Run the Setup and install.
4.Javafx Setup on your machine.
    4.1 Install javafx jdk from gluon website.
    4.2 Set the JAVA_HOME system variable to the /bin folder of your installation
5.Maven setup 
    5.1 Install maven binaries from the website.
    5.2 Add /bin path of installation to the PATH variable.
    5.3 Run mvn -version to see if maven is correctly configured.
6.VScode setup
    6.1 Install vscode from the website.
    6.2 Install java extension pack from the vscode extension store.
    6.3 Install javafx support extension from the vscode extension store.
    6.4 Install maven support extension from the vscode extension store.
    6.5 Install mysql support extension from the vscode extension store.
    6.6 Install scenebuilder support extension from the vscode extension store.
    6.7 Install css support extension from the vscode extension store.


-------------------------------------------------------------------------------------------


-------------------------------------------------------------------------------------------
------------------------------------Steps to execute Project-------------------------------
-------------------------------------------------------------------------------------------

1. Install the jar file into your system.
2. Make the mysql db by running the db script.
3. Change the username and password to your own as required in code in the Dbcontroller.java file.
4.Open a terminal and browse to the jars directory.
5. Use the following command to run the jar
    java -jar BankManagementSystem.jar
